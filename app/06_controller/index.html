Notes:

Ng-controller does 3 steps:

  • Create a child scope: $scope = $parent.$new(false);
  • Instantiate a new controller of type MyController: $scope.$ctrl = $controller('MyController');
  • Compile all the DOM under ng-controller with the $scope: $compile($element.contents())($scope);
